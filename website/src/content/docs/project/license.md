Dispatch is **commercial, proprietary software** - © 2026 CinderHills, all rights reserved. It is not open
source. Running Dispatch requires a valid **license key**.

## How licensing works

- **Offline, no call home.** A license key is validated **locally** against a public key embedded in the
  product. Nothing about your install or your mail is ever sent to us - see [Security](/security/).
- **Node-locked.** Each key is issued for one install's **Machine ID** (shown on the dashboard's **License**
  page). A key only works on the install it was issued for.
- **Grace period.** A new install runs for 30 days so you can obtain a key; after that, an unlicensed install
  stops accepting and relaying new mail until a valid key is entered. Your spool and dashboard stay up, so
  entering a key immediately restores normal operation - nothing queued is lost.

## Getting a license

Enter your license key on the dashboard under **System → License**. That page shows the Machine ID to provide
when you purchase or renew. This page is a summary, **not** a contract; the license terms you agree to at
purchase govern.
