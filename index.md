## What it does

- Accepts mail over **SMTP** or a **Mailgun-compatible HTTP API**
- Forwards to **Mailgun, SendGrid, Amazon SES, Postmark, Resend, SparkPost, SMTP2GO, Azure**, or any SMTP host
- **Durable spool** — mail keeps flowing even if the database is down
- **Live dashboard** — counters, searchable message log, and provider testing
- **One place** to manage credentials and watch every message
